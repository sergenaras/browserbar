## Installation
No need to build from source!

1. Go to the **[Actions](https://github.com/sergenaras/browserbar/actions)** tab.
2. Click on the latest workflow run.
3. Download the **BrowserBar-App** artifact.
4. Unzip and run!

> Note: Since this is an open-source app, you may need to right-click and select "Open" the first time.
